Steamerjs 介绍

steamerjs 介绍

Plugin is command, use when you plugin in.

插件即命令，即插即可用。

安装

// 安装核心库
npm i -g steamerjs

// 安装命令插件
npm i -g steamer-plugin-xxx

使用

steamer xxx

// alias command
str xxx

更新

// 更新本身
npm update -g steamerjs

// 更新插件
npm update -g steamer-plugin-xxx

// 更新脚手架
npm update -g steamer-xxx
