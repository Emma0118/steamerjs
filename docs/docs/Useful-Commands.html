Useful Commands 开发命令

Useful Commands 开发命令

使用命令

// 安装依赖
npm i

// 开发
npm start 或 npm run dev
// 打开链接
localhost:9000

// 查看开发环境源码
npm run source

// 代码规范安装
npm i -g eslint
npm i -g stylelint
// 代码规范扫描
npm lint

// 生产代码生成
npm run dist 或 npm run pub
