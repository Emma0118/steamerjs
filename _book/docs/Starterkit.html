Starterkit 脚手架

Starterkit 脚手架

Steamer目前主要通过 steamer-plugin-kit 进行脚手架的管理（安装、更新等）。目前官方有以下脚手架：

  • steamer-example
    • 脚手架规范
  • steamer-simple
    • 一个基础而无框架的脚手架
  • steamer-react
    • react 脚手架
  • steamer-react-component
    • react 组件开发脚手架
  • steamer-vue
    • vue 脚手架
